01

How it works

It scrapes Reddit live for real posts on your topic, then uses those as examples. The model sees what real humans wrote — not AI imitations — before rewriting your text.

02

5-pass pipeline

Topic extraction → Reddit scraping → Rewrite → Surgical scrub → Verification. Each pass catches what the previous missed. The verification pass has the model check its own output.

03

Set your writing style

Paste a sample of your real writing before running. Even a few sentences makes a big difference — it extracts your rhythm and applies it as hard rules.

04

Make a few edits yourself

Change 2–3 sentences to exactly how you'd write them. Genuinely human text that no AI can replicate. Biggest single improvement you can make.

05

Burstiness score

Human writing scores 0.5+. Below 0.4 means sentence lengths are too uniform — GPTZero's #1 signal. The pipeline enforces this but check the score after.

06

Running the server

This tool needs the Node server running. Open terminal in the project folder and run: npm start
