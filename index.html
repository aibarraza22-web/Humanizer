Tip 01

Set your writing style first

Paste a sample of your real writing before humanizing. Even 5 sentences from an email picks up your rhythm. Single biggest improvement you can make.

Tip 02

Max mode targets detectors

Max mode rebuilds sentence structure, injects fragments, forces specific word substitutions, and adds deliberate imperfections. Balanced mode is for when you need less change.

Tip 03

Make 2–3 edits yourself

After humanizing, change 1–2 sentences to exactly how you'd write them. Genuinely unpredictable patterns no AI can produce. Takes 30 seconds.

Tip 04

Split long texts

Over 700 words? Split into sections and humanize each one separately. Quality drops noticeably on longer chunks.

Tip 05

Run it twice

Humanize → copy → paste back → run on Subtle mode. Two passes creates layered variation that's hard to fingerprint as any known humanizer pattern.

Tip 06

Burstiness score

Human writing scores 0.5+. Below 0.4 means sentence lengths are still too uniform — the #1 detection signal on both Turnitin and GPTZero. Run again if you see red.

Tip 07

Datasets = reversed GPTZero

Every run pulls fresh Reddit writing via JSON API, builds AI lookalikes from those exact topics, and compares both in the prompt so the model follows real human patterns instead of static canned samples.

Tip 08

Fragment injection

The scrubber injects deliberately incomplete sentences — fragments. Real humans write fragments constantly. AI almost never does. Each fragment is a perplexity spike that detectors read as human.
