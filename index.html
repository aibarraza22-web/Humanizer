01

Use it like ChatGPT

The Chat tab works like a normal AI assistant. Ask it anything — questions, code, opinions, essays. The difference is every answer sounds like a real person wrote it, not an AI.

02

Human writing, not AI paraphrasing

The Humanize tab doesn't just swap synonyms — it rewrites from scratch using patterns from real human writing. Plus a quality gate that runs a second cleanup pass if the first rewrite still has AI patterns.

03

Your own Mistral key

You bring your own Mistral API key so you pay at cost, not markup. Mistral Large is fast and cheaper than GPT-4. Get your key at console.mistral.ai.

04

The quality gate

After the first rewrite, the pipeline scores the output for AI patterns. If it finds banned words, AI openers, or hedging phrases, it runs a targeted cleanup pass. This is what makes it reliable.

05

Make 2–3 edits yourself

The single biggest upgrade: change a few sentences to exactly how you'd write them. No AI can replicate that. It's what separates passable from undetectable.

06

Running the server

Open terminal in the project folder and run: npm start
