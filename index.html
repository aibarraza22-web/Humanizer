01

Trained on real data

The pipeline uses a real human vs AI dataset plus 6 real answers as voice examples. It knows what human writing looks like — it's not guessing at "casual".

02

Iterative fix loop

Load examples → Analyze → Rewrite → 3 rounds of targeted fixes. Each fix round finds what's STILL detecting as AI and fixes only those things — long sentences, dramatic phrases, essay structure, uniform rhythm.

03

Answer as Me

Use the Answer tab for discussion posts, short answers, or any question you need to respond to. It answers the way you'd answer — direct, confident, no hedging.

04

Make a few edits yourself

Change 2–3 sentences to exactly how you'd write them. Genuinely human text no AI can replicate. The single biggest improvement you can make.

05

Works on research papers too

Paste sections of a research paper — the pipeline preserves all your citations, evidence, and arguments while rewriting the prose to sound human.

06

Running the server

Open terminal in the project folder and run: npm start
