Tip 01

Set your writing style first

Paste a sample of your real writing before humanizing. Even 5 sentences from an email picks up your rhythm. The single biggest improvement you can make.

Tip 02

Max mode for detectors

Balanced mode is for clean-ups. Max mode fully rebuilds sentence structure and rhythm — which is what actually moves your score on Turnitin and GPTZero.

Tip 03

Make 2–3 edits yourself

After humanizing, change 1–2 sentences to exactly how you'd say them. This injects genuinely unpredictable patterns that no AI rewriter can produce.

Tip 04

Split long texts

For anything over 700 words, split into 2–3 sections and humanize each separately. The model focuses better on shorter chunks.

Tip 05

Run it twice

Humanize → copy output → paste it back → run on Subtle mode. The second pass adds variation that's very hard for detectors to flag.

Tip 06

Check burstiness score

Human writing scores 0.5+. Below 0.4 means sentence lengths are still too uniform — the #1 AI detection signal. Run again if needed.

Tip 07

How the datasets work

Every run, 6 real human paragraphs and 6 real AI paragraphs plus 3 same-topic contrast pairs are randomly selected. The model is shown all of them and told to write toward one, away from the other.

Tip 08

Quiz answers are hard rules

Quiz answers become ABSOLUTE rules enforced separately from the rewrite instructions. If you said never start with "And" — every sentence gets checked before output.
