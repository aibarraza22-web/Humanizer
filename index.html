Watch the burstiness score

Human writing scores 0.5+. Below 0.4 means sentence lengths are too uniform — the #1 GPTZero signal. Run again on max mode if you see red.

06

Split long texts

Over 700 words? Split into sections, humanize each separately. Quality drops on long chunks and API costs climb fast.
